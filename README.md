## Objective

Interactive comprehensive music theory assistance tool for guitarists.

## Development workflow

### Setup

    brew install node
    npm install -g pug-cli

### Dev environment

    pug --watch *.pug -o ./
    python2 -m SimpleHTTPServer
    python3 -m http.server
